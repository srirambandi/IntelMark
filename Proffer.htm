Products
Smart Contracts
Proffer.sol
Stores a hash of the latest verified expertise data, holds token stakes during search, performs token rewards and penalties.
ProfferToken.sol
ERC20 token contract tracking balances of Proffer users. The token is essentially a product license, as all all actions on Proffer are staked with the token.
JS Libraries
ReadExpertise.js
Allows any app or user to query expertise of Proffer users by user account address or topics. Expertise is stored off-chain on IPFS but verified on-chain.
ExpertiseMiner.js
Updates to Proffer expertise data occur only through consensus among Expertise Verifiers, who execute this script. Details in Section 6 of the whitepaper.
